##Steps for adding external jars in IntelliJ IDEA:
+ Click File from File menu.
+ Project Structure ( CTRL + SHIFT + ALT + S on Windows/Linux, ⌘ + ; on Mac OS X)
+ Select Modules at the left panel.
+ Dependencies tab.
+ Add... → Project Library → Attach Jar.
